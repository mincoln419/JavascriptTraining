## 2. 게시판 만들기
기본적인 CRUD 기능을 할 수 있는 게시판


<페이지 목록>
- 목록 페이지
  * 데이터 목록을 조회
  * 등록 버튼을 눌렀을 경우 신규 게시글을 작성할 수 있는 상세페이지로 이동
- 상세 페이지
  * 데이터 목록의 상세데이터를 조회
  * 상세데이터를 등록, 수정, 삭제 가능
  * 등록 버튼은 신규 등록화면에서만, 수정/삭제 버튼은 특정 상세 데이터에 대해서만 노출
  * 등록, 수정, 삭제 처리후 메인페이지로 이동
